# Category Playbooks

Use this file when the default 7+1 image structure is too generic. Pick the closest category and adapt the image set accordingly.

## 1. Tools / Hardware / Garage

Buyer priorities:
- fit
- durability
- installation clarity
- kit completeness

Images that matter most:
- size/components
- compatibility/installation
- detail/material
- problem/solution

Good scenes:
- product mounted, attached, tightened, stored, or used in a work task

Avoid:
- luxury lifestyle staging
- soft decorative backgrounds with no task logic

## 2. Home / Organization / Kitchen

Buyer priorities:
- where it goes
- how much it holds
- whether it saves space
- whether it looks tidy in use

Images that matter most:
- primary use case
- storage/package
- problem/solution
- size/context

Good scenes:
- real countertop, cabinet, pantry, drawer, closet, or bathroom use

Avoid:
- generic living-room glamour shots that do not explain function

## 3. Beauty / Personal Care

Buyer priorities:
- comfort
- hygiene
- texture / finish
- how it is used on body or face

Images that matter most:
- use case
- detail/material
- package/components
- problem/solution

Good scenes:
- clean personal-care environment with believable usage

Avoid:
- exaggerated claims
- medical-looking claims without support

## 4. Pet Products

Buyer priorities:
- fit for pet size
- safety and comfort
- ease of use for owner
- durability / cleaning

Images that matter most:
- primary use case
- size/fit
- detail/material
- problem/solution

Good scenes:
- realistic pet behavior in home, car, or walking context

Avoid:
- impossible pet poses
- unsafe usage

## 5. Automotive / Bike / Accessory

Buyer priorities:
- compatibility
- installation
- durability
- where and how it mounts

Images that matter most:
- compatibility/installation
- detail/material
- problem/solution
- size/components

Good scenes:
- mounted on the real class of vehicle or tool it belongs to

Avoid:
- ambiguous mounting logic
- vehicle scenes where the product is too small to read

## 6. Electronics / Desk Accessories

Buyer priorities:
- interface clarity
- compatibility
- cable / port / attachment logic
- workflow improvement

Images that matter most:
- primary use case
- compatibility
- detail/material
- problem/solution

Good scenes:
- desk, workstation, laptop, phone, charging, audio, or travel use

Avoid:
- overly abstract neon-tech scenes that hide the real product

## Selection Rule

If the product spans multiple categories, choose the category that best matches the buyer's purchase objection, not just the catalog taxonomy.

Examples:
- a cable organizer sold for kitchen drawers belongs closer to home/organization than electronics
- a dog-car-seat tether belongs closer to pet products than automotive
