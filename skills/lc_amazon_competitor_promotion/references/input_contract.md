# 输入契约

## 用户需要准备什么

用户只需要准备一个本地文件夹，里面放西柚找词下载的“流量得分趋势”Excel。

每个 Excel 对应一个竞品 ASIN。当前支持 1-10 个 ASIN。

不需要用户提供：

- 除西柚“流量得分趋势”Excel 文件夹之外的其它数据或配置。
- zip 压缩包。

## 目录规则

输入入口是目录：

```text
case_dir/
  流量得分趋势_US_B0FRLWC6X9_20250618to20260618.xlsx
  流量得分趋势_US_B0FP51DN7C_20250618to20260618.xlsx
```

工具会忽略：

- `__MACOSX/`
- `._*`
- `~$*`
- 非 `.xlsx` 文件

## 必需字段

每个数据 sheet 第一行必须包含：

| 字段 | 用途 |
|---|---|
| `日期 (数据来源于西柚洞察)` | 日维度时间轴 |
| `ASIN` | 竞品 ASIN |
| `综合流量得分` | 总流量背景 |
| `自然流量得分` | 自然流量有效性评估 |
| `广告流量得分` | 广告放量识别 |
| `自然排名流量得分` | 自然排名流量背景 |
| `SP流量得分` | 广告结构分项 |
| `SOR流量得分` | 广告结构分项 |
| `SB流量得分` | 广告结构分项 |
| `SBV流量得分` | 广告结构分项 |
| `AC流量得分` | 广告结构分项 |
| `OOR流量得分` | 广告结构分项 |
| `价格($)` | 价格和最后成交价计算 |
| `Deal($)` | Deal 动作识别，不参与最后成交价计算 |
| `划线价格($)` | 价格辅助展示 |
| `Prime价格($)` | 价格辅助展示 |
| `Ratings评分` | 评分异常识别 |
| `Ratings数量` | 评论突增和评价异常识别 |
| `新增广告活动` | 广告架构变化识别 |
| `Coupon` | Coupon 动作和最后成交价计算 |
| `Promotion` | 折扣动作和最后成交价计算 |
| `Subscribe` | Subscribe 动作和最后成交价计算 |
| `Other` | 其他促销和最后成交价计算 |

`BSR[...]` 字段属于动态列，可能随类目变化，不作为必需固定列，但识别到时应保留。

## 数据口径硬约束

- 价格、成交价、划线价、Prime 价、Coupon、Promotion、Subscribe、Other 只来自西柚 Excel。
- Ratings 评分和 Ratings 数量只来自西柚 Excel。
- 新增广告活动和广告流量分项只来自西柚 Excel。
- 自动补充数据返回的价格、折扣、rating、reviews 不得用于价格/促销/评价规则。
- 自动补充数据只用于 Woot、buyBox、销量预测、上架时间、类目/季节性等补充能力。

## `inspect-inputs` 输出

`ready` 条件：

- 识别到 1-10 个有效西柚流量趋势 Excel。
- 所有 Excel 必需字段齐全。
- 所有 Excel 内部 ASIN 可识别。
- 所有 Excel 站点一致。
- 每个 Excel 至少有 14 天有效日期。

`needs_confirmation` 条件：

- 文件数量超过 10。
- 多站点混杂。
- 某些文件缺字段。
- Excel 内部 ASIN 与文件名 ASIN 不一致。
- 某个文件有效日期不足。

`failed` 条件：

- 没有识别到有效 `.xlsx`。
- 所有 `.xlsx` 都不是西柚流量趋势表。
