# 计算口径

## 清洗

- 主分析样本：有 ASIN、价格 > 0、月销量 > 0。
- 价格口径：`价格($)`。
- 销量口径：`月销量`。
- 销售额口径：`月销售额($)`。
- 缺失评分数 / 评分：按 0。
- 明显不相关产品：先由 CLI 输出全量 `agent_relevance_workspace.json`，再由 Agent 对每个 ASIN 标 `related` / `irrelevant` / `uncertain`；只按 `relevance=irrelevant` 导出的 `exclude-asins` ASIN 清单剔除，不由 CLI 自动猜测。
- 清洗后近 30 天 listing 数 < 50：标记低置信度。
- 如果输入表行数正好为 2000 或 3000，标记“可能达到卖家精灵导出上限”；所有占比仍以实际导出行数为分母。

## 总指标

- 年总销量：12 个月竞品月表清洗样本的 `月销量` 汇总。
- 年总销售额：12 个月竞品月表清洗样本的 `月销售额($)` 汇总。
- 近 30 天平均售价：近 30 天清洗样本 `价格($)` 平均。
- 核心词近 90 天平均转化率：保留词 `近90天购买量` 汇总 / `近90天点击量` 汇总。
- 核心词加权平均 PPC：每个词 `PPC竞价-推荐($)` * 该词点击量占比 后求和。
- 市场广告压力 ACOS：加权平均 PPC / (平均转化率 * 平均售价)。

## 垄断指数

头部比例为前 10%。

- 链接垄断：按 ASIN/listing 的月销量降序取前 10%。
- 品牌垄断：按品牌聚合月销量，降序取前 10% 品牌。
- 卖家垄断：按 BuyBox 卖家或卖家信息聚合月销量，降序取前 10% 卖家。

输出销量占比、销售额占比、平均价格；品牌和卖家额外输出链接数量。

## 新品推新成功率

- 新品：`上架天数 <= 360`。
- 成功头部：当月销量进入全样本前 20%。
- 成功率：新品中进入成功头部的 listing 数 / 新品 listing 数。

12 个月趋势按每个月当时在榜集合单独计算。

## 基础透视

每个透视输出：

- Listing 数量占比
- Listing 销量占比
- Listing 数量
- 平均销量

维度：

- 评论数区间
- 上架天数区间
- 评分区间
- 售价区间（按近 30 天平均售价倍数）

每个区间样本数目标为至少 10；不足时顺序合并相邻区间。

## 规则结论

- 评论依赖性：按评论数降序取前 10% 商品销量占比。
- 评分值依赖性：评分区间由低到高，找第一个销量占比 > listing 数量占比的区间。
- 价格敏感度：按价格升序取前 10% 商品销量占比。
- 最值得进入的价格带：先筛选销量占比 > listing 数量占比的价格带，再按“销量占比 / listing 数量占比”的倍数倒序取前 2 个。
- 新品打造周期：上架时间区间由低到高，找第一个销量占比 > listing 数量占比的区间。
- 商品垄断度：按销量降序取前 10% 商品销量占比。
- 品牌垄断度：按品牌销量降序取前 10% 品牌销量占比。
- 市场广告压力指数：按 ACOS 区间打分。

份额类等级：

- `>= 80%`：极高
- `>= 60% 且 < 80%`：很高
- `>= 50% 且 < 60%`：较高
- `>= 40% 且 < 50%`：中等
- `>= 30% 且 < 40%`：较低
- `>= 20% 且 < 30%`：很低
- `< 20%`：极低

ACOS 等级：

- `>= 100%`：极高
- `>= 80% 且 < 100%`：很高
- `>= 60% 且 < 80%`：较高
- `>= 40% 且 < 60%`：中等
- `>= 25% 且 < 40%`：较低
- `>= 15% 且 < 25%`：很低
- `< 15%`：极低

## 头部品牌/卖家外部调研

- 调研对象：从清洗后近 30 天品牌榜、卖家榜各取前 10；由 `external-targets` 从已生成报告中导出。
- 调研方式：Agent 联网搜索公开信息，并按固定 JSON 结构落盘。
- 输出内容：头部品牌全域流量布局评估、品牌背后公司实力评估、市场级外部竞争强度总结。
- 结论性质：公开信息推断，不是尽调结论；不可推翻结构化市场数据，只能辅助解释竞争强度。
- 不参与重算：不会改变总指标、垄断度、基础透视、8 项规则结论和 Top90 覆盖。
